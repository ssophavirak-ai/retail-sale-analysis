# Retail Sales Performance & Executive Insight Dashboard

## 📖 Project Overview
This project is an interactive Power BI dashboard designed to analyze retail sales performance and provide executive-level business insights. The dashboard helps identify revenue trends, regional performance, product sales, and top-performing companies to support data-driven decision making.

---

## 📊 Dataset
The dataset contains retail sales transaction data, including:
- Revenue
- Orders
- Products
- Companies
- Sales Regions
- Monthly Sales

---

## 🛠️ Tools Used
- Power BI Desktop
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
- Microsoft Excel (Data Preparation)

---

## 💡 Key Insights
- Generated 3.06M in total revenue from 43K orders.
- The North and South regions generated the highest revenue.
- Vertex Retail was the top-performing company.
- Tea and Strawberries were the highest-selling products.
- Sales reached their highest level in November, showing a strong seasonal peak.
